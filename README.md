## Clone & Run

```bash
# clone
git clone https://github.com/a2rp/sde-qna-handbook.git
cd sde-qna-handbook

# install deps
npm install

# start dev server
npm run dev

# build for production
npm run build

# preview the built app
npm run preview
```
